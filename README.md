# CS2-ExternalCheat
Nowadays, there is a constant stream of cheats at the CS2, with most of the cheat features already implemented. Interesting aspect that I came across during my research is external silent aim. Of course, silent aim is easily achievable after injecting into the game you just need to hook the CreateMove function to intercept the CMD structure data.
